I am a programmer interested in code generation across multiple languages from high level diagrams. The pages contain details about how to bootstrap a self-generating code generation system from it's own design diagrams. Currently I am using the ideas in this project to generate configuration (aka code) for datacenter networks using Cumulus Linux.
